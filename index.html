My first Webpage Update Page

This page will go through some different types of Linux,Vi, and Git Commands. Each command will have a description on its function.

Linux Commands

- CD- Change Directory

- Mkdir Name- Make Directory

- Ls- List Files

Vi Commands

- Vi Filename- replace with the actual file name

- Vi Commands-

I- insert mode

Escape- get out of insert mode

Shift Key + colon- command prompt for vi

W- command to write

Q- command to quit

WQ- write and quit

Git Commands

- Git init- create the repository

- Git Add filename- add the file to git in preparation for a commit

- Git commit -m message - do the commit to save changes

- Git checkout -b branch name- make a new branch and checkout

- Git checkout branch name- checkout branch

- Git merge BRANCHNAME- merge the branch, be on the branch that you want to merge to
